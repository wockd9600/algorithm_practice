컴퓨터적인 사고방식의 적응과 코드를 더 효율적이고 빠르게 작성하기 위해서
(해당 깃의 모든 프로그래밍 언어는 python이다.)


### 백준
해당 폴더는 아래의 이유 중 하나로 저장한다.
* 내가 몰랐거나 미숙한 코딩 방법, 라이브러리
* 한 번 더 보고 싶은 코딩


### 이것이 코딩테스트다(개인)
